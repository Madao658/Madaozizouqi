# 异界牌桌自走棋原型

这是一个纯静态网页原型，当前版本已经包含棋池、商店、备战区、棋盘布阵、羁绊显示和简化自动战斗。
游戏开局会显示「MadaoFamily乱斗自走棋」欢迎幕；第 25 回合是最终回合，最终战胜利即通关。

## 打开方式

直接用浏览器打开：

```text
D:\work\codex-all\zizouqi\index.html
```

不需要安装依赖，也不需要启动服务器。

## 当前玩法

1. 点击商店里的棋子购买。
2. 把备战区棋子拖到棋盘下半区进行上阵；拖到已有棋子的格子会交换位置。
3. 棋盘和备战区之间也可以互相拖拽，点击选中再点击目标格仍然可用。
4. 点击「开始战斗」生成一组敌人并播放战斗。
5. 战斗结算后点击「进入下一回合」获得金币、经验、刷新商店并继续。

## 已实现

- 20 个棋子。
- 6 个阵营羁绊。
- 6 个职业羁绊。
- 三合一升星，最高 3 星。
- 等级和经验：玩家从 2 级 0 经验开始，等级决定可上阵棋子数量。
- 金币、生命、回合。
- 随机商店与刷新。
- 花费 4 金购买 4 经验。
- 拖拽布阵与棋子交换。
- 简化自动战斗模拟。
- 可见战斗播放：棋子会在棋盘上移动、攻击、掉血并显示受击反馈。
- 战斗日志。
- 开局欢迎弹窗。
- 第 25 回合最终战和通关胜利弹窗。

## 数据文件

- `data/units.json`：更完整的棋子和羁绊设计数据。
- `src/app.js`：当前网页原型使用的紧凑运行时数据。

后续可以把 `src/app.js` 改成从 `data/units.json` 读取；如果继续保持纯文件打开模式，则需要把 JSON 数据嵌入页面或启动一个本地静态服务器。

## 等级经验

- 初始：2 级，0 经验。
- 初始金币：4。
- 上阵上限：等于当前等级。
- 回合结束：自动获得 2 经验。
- 回合结束金币：5 基础金币 + 利息。
- 利息：每拥有 10 金币多获得 1 金，最多 5 金；50 金及以上达到最大利息。
- 购买经验：花费 4 金获得 4 经验。
- 升级经验：2->3 需要 2，3->4 需要 6，4->5 需要 10，5->6 需要 14，6->7 需要 20，7->8 需要 36，8->9 需要 48，9->10 需要 84。
